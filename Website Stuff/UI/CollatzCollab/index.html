What is Collatz?

The Collatz Conjecture is the most common name for a mathmatical conjecture discovered a long time ago.
The conjecture goes as this: Pick any positive number 1 - ∞ . If the number is even, divide it by 2, if it's odd, multiply it by 3 and add 1. Heres an  Example:  x = 10, 10's even so 10/2=5, 5 is odd so (5*3)+1=16, thats even, 16/2=8, 8/2=4 here's the start of the loop... 4/2=2, 2/2=1, (1*3)+1=4. There you go, the conjecture states that every positive integer imaginable will eventually come back to 1 and loop back to 4 for ever. As of right now, there hasn't been a number tested that hasn't eventually come back to the 1 and 4 loop.

What  is the Collab?

The Collax Collab is a program that runs descretly on your computer, the program is sent a number by our server and then the program runs the number according to the rules of the conjecture. For as long as this conjecture has existed, people have been trying to solve it by hand, but why? Computers are way faster and due to the simplicity of the problem we can run numbers though the program in fractions of a second. The idea of the program is to allow multiple different computers to run numbers together sort of like a Bot Net. 

What  is the goal?

The goal of this project is to either prove that every number works with this conjecture, or to find a number that doesn't work with it.

What  if I reach a goal?

If your computer finds a number that proves the conjecture wrong, then you get prize! The first computer to find a conflicting number gets a cash prize (price to be determined).  Thats literally free money.

What  does it cost?

The program is completly free. The program also is extremely light on your computer, meaning it isn't going to completly slow down your computer.
